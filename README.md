# cover-letter-generator

Full-stack TypeScript project.

## Structure

- `package.json` — root scripts for running the full stack
- `backend/` — Node.js + Express + TypeScript API
- `frontend/` — React + Vite + TypeScript UI

## Development

### Start everything

```bash
npm run dev
```

This starts:
- the backend on http://localhost:3001
- the frontend on http://localhost:5173

### Start apps individually

#### Backend

```bash
cd backend
npm run dev
```

#### Frontend

```bash
cd frontend
npm run dev
```
